SeatMaster Updates

A timeline of new features, visual improvements, and major changes.

v1.0 – Initial Prototype

First working version built in Code.org App Lab:

  • Basic floor map buttons
  • Open / occupied toggle system
  • Simple counters for table availability

v1.1 – List Mode & Filters

Expanded functionality with a new list view:

  • List Mode with seat counts
  • Filters for open, occupied, booths, indoor, patio
  • Cleaner color coding for readability

v1.2 – Demo Login Screen

A basic login gate for class demonstrations:

  • Username: Admin
  • Password: SeatMaster2026
  • Prevents random table editing during demos
